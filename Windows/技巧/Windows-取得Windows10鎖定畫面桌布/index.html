Windows - 取得 Windows 10 鎖定畫面桌布

Posted on 2018-08-07 In Windows , 技巧

有時候剛打開 Windows 停留在鎖定畫面，會發現鎖定畫面的桌布超級好看，可是網路上又找不到相同的圖片，那就只好從 Windows 的深層挖出來啦

首先打開你的檔案總管，並且導航到C:\Users\你的名字\AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets

接著將資料夾以檔案大小排序，並且將最大最肥的那幾個檔案複製出來

然後把副檔名改為 .jpg 之後，你就獲得超好看的桌布啦!
